Project Overview

This project automates AWS infrastructure provisioning using Python and boto3.

The automation creates and manages:

- Amazon S3 Buckets
- Amazon EC2 Instances
- AWS Resource Inventory
- Resource Cleanup Operations

The goal is to reduce manual AWS Console work and demonstrate Infrastructure as Code (IaC) concepts.
